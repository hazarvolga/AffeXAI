# Aluplan Backend

This is the backend for the Aluplan CMS system, built with NestJS.

## Features
- CMS API for managing pages and components
- User authentication and authorization
- Email marketing system
- Event management
- Media management
- Notification system

## Getting Started

\```bash
# Install dependencies
npm install

# Run in development mode
npm run start:dev

# Run tests
npm run test
\```
