# Modern Landing Page Builder Components - Comprehensive Prompt Guide

## 1. HERO SECTIONS (MODERN LANDING PAGE)

**Component Purpose**: Make an unforgettable first impression with bold visuals and clear value proposition that stops scrolling

**Required Variations**:
- Full-screen animated gradient hero with 3D floating elements
- Split-screen hero (interactive product demo + benefit-focused copy)
- Video background hero with text overlay and glassmorphism card
- Minimal typography-focused hero with kinetic text animations
- Bento-grid hero (multi-panel mosaic with micro-interactions)
- Immersive WebGL/Three.js hero with particle effects

**Design Specifications**:
- Height: 100vh (full viewport), no compromise on impact
- Typography: Headline 64-96px (ultra-bold, display font), subhead 24-32px (light/regular)
- Gradient Trends: Animated mesh gradients, holographic effects, dual-tone splits
- 3D Elements: Floating product mockups, abstract geometric shapes with depth
- Glassmorphism: Frosted glass effect cards (backdrop-filter: blur(10px), rgba backgrounds)
- Animation: Parallax scroll (foreground moves faster than background), fade-in stagger
- Interactive Elements: Mouse-follow effects, cursor-triggered animations, scroll-velocity reactions
- CTA Design: Oversized buttons (min 60px height), pill-shaped, with glow/shadow effects
- Typography Effects: Gradient text fills, animated underlines, typewriter effects, morphing text
- Color Psychology: Bold, saturated primaries for tech/SaaS, pastels for creative/wellness
- Micro-interactions: Hover states trigger 3D transforms, buttons breathe/pulse subtly
- Loading State: Skeleton screens or elegant loading animations (not spinners)
- Responsive: Maintain drama on mobile, reduce animation complexity for performance
- Accessibility: Reduced motion alternative, 4.5:1 contrast maintained despite effects

**Modern Trends to Incorporate**:
- Neumorphism subtle touches (soft shadows, embossed effects)
- Claymorphism (3D clay-like textures and shadows)
- Dark mode by default with light mode toggle
- Asymmetric layouts that break the grid intentionally
- Oversized typography that bleeds off screen

- Grid Layout: Asymmetric bento boxes, not uniform grids (mix 1x1, 1x2, 2x1 cells)
- Color Accents: Each feature gets unique accent color (purple, blue, green, pink, orange)
- Video Clips: 6-10 second looped demos, no sound, auto-play on hover/scroll
- Scroll Animations: Features fade-in and slide-up sequentially (100ms stagger)
- Interactive Elements: Toggle between features, tabs with smooth transitions
- Comparison UI: Draggable slider handle, clear before/after labels, image overlays
- Background: Subtle animated gradients, noise textures, or radial gradient spotlights
- Spacing: 40-60px gaps between cards, 120-160px section padding
- Responsive: 3-column â†’ 2-column â†’ 1-column, maintain card aspect ratios
- Accessibility: Keyboard-navigable cards, reduced motion mode removes complex animations

**Modern Design Language**:
- Brutalist touches: Thick borders, bold type, intentional "raw" elements
- Maximalist color: Don't shy from vibrant, high-saturation colors
- Depth through shadows: Multiple shadow layers for realistic depth
- Organic shapes: Blob shapes, liquid morphs, abstract forms

## 3. SOCIAL PROOF / TESTIMONIAL SECTIONS

**Component Purpose**: Build trust through authentic user voices with modern presentation that doesn't feel stock or templated

**Required Variations**:
- Infinite horizontal scroll testimonial ticker (auto-scrolling)
- Video testimonial grid with play-on-hover
- Tweet-style testimonial cards (social media aesthetic)
- Animated rating showcase with counting numbers
- Marquee-style logo parade (clients/press)
- Interactive testimonial carousel with 3D perspective

**Design Specifications**:
- Card Styling: Modern shadows, 20px+ border radius, white or dark cards with high contrast
- Profile Photos: 56-72px, perfect circles, subtle border (2px) with brand color
- Quote Typography: 20-24px, medium weight, quotation marks stylized or removed
- Attribution: Name 16-18px bold, title/company 14-16px, lighter color (60% opacity)
- Star Ratings: Large stars (28-32px), animated fill-in on view, gold/yellow (#FFB800)
- Infinite Scroll: Seamless loop using CSS animations or JS, 20-40s duration
- Video Thumbnails: 16:9 ratio, play button overlay (64px), thumbnail with slight motion
- Social Media Style: Twitter/LinkedIn UI mimicry (verified badges, timestamps, interaction icons)
- Number Counters: Animate from 0 to value on scroll-into-view (4.9/5, 10K+ users)
- Logo Treatment: Grayscale logos with smooth color transition on hover
- Marquee Speed: Slow enough to read (30-40px/second), pause on hover
- 3D Carousel: Cards rotate on Y-axis, perspective depth (1000-2000px)
- Background: Subtle gradient or solid color different from main page background
- Grid Variations: Mix card sizes (some 1x, some 2x width for featured testimonials)
- Responsive: Infinite scroll maintains on mobile, carousel becomes swipeable
- Accessibility: Pause animations button, keyboard navigation, screen-reader friendly

**Trust Elements to Include**:
- Real photos (no stock images - users can tell)
- Specific details (numbers, outcomes, timelines)
- Video preferred over text when possible
- Verified badges for authenticity
- Company logos that are recognizable

## 4. CALL-TO-ACTION SECTIONS

**Component Purpose**: Convert visitors with high-impact, impossible-to-miss CTAs that create urgency and excitement

**Required Variations**:
- Full-bleed gradient CTA with animated elements
- Floating 3D card CTA with motion graphics
- Split CTA (two options: primary vs secondary)
- Sticky bottom-bar CTA that reveals on scroll
- Exit-intent popup with special offer
- Gamified CTA (scratch-off, spin-wheel, interactive element)

**Design Specifications**:
- Button Size: Hero CTAs 64-72px height, secondary 52-60px, min 200px width
- Button Style: Gradient backgrounds, glow effects, animated gradient shifts
- Hover States: Lift with shadow, scale 1.05x, glow intensity increase, gradient animation
- Typography: Button text 18-20px, bold weight, uppercase or sentence case
- Icons in Buttons: Arrow (â†’), sparkle (âœ¨), rocket (ðŸš€) 24px, positioned right
- Background Treatment: Vibrant gradients (purple-to-pink, blue-to-cyan), animated mesh
- Urgency Elements: Countdown timer (48-60px numbers), "Limited spots" badges
- Social Proof: "Join 50K+ users" text above button, small logos below
- Form Integration: Email capture input inline with button, single-line layout
- Multiple CTAs: Primary button prominent, secondary button ghost/outlined style
- Sticky Bar: 70-80px height, shadow on top, background blur, Z-index 1000
- Exit Intent: Centered modal, 600-800px width, attractive offer, easy close (X)
- Gamification: Wheel 300-400px, bright colors, clear prize labels, spin animation
- Animation: Pulse effect (2s loop), shimmer across button, gradient rotation
- Micro-copy: "No credit card required", "14-day free trial", "Cancel anytime"
- Responsive: Full-width buttons on mobile, maintain button hierarchy
- Accessibility: High contrast (7:1 for AAA), focus rings, clear disabled states

**Conversion Optimization**:
- Action-oriented copy: "Start creating" not "Sign up"
- Remove friction: No multi-field forms on first CTA
- Visual hierarchy: One clear primary action
- Trust signals: Security badges, money-back guarantee

## 6. PRICING SECTIONS

**Component Purpose**: Present pricing with modern design that makes decision-making easy and highlights value

**Required Variations**:
- 3-tier comparison cards with featured tier elevated
- Toggle between monthly/annual with savings highlight
- Interactive pricing calculator (sliding scale by usage)
- Single CTA pricing (one product, multiple add-ons)
- Comparison table with expandable feature details
- Enterprise custom pricing with contact form

**Design Specifications**:
- Card Design: 380-420px width, elevated shadows on hover, 20-24px radius
- Featured Plan: Scale 1.05x, "Most Popular" badge, different shadow/glow
- Typography: Plan name 28-32px, price 56-72px (large and bold), /month 18-20px
- Price Display: Emphasis on number, de-emphasize currency symbol (smaller/lighter)
- Savings Badge: "Save 20%" pill badge in green, positioned top-right of card
- Toggle Switch: iOS-style toggle, 60-80px wide, smooth animation (300ms)
- Feature List: Checkmarks (24px, green), X marks (24px, gray/red), 16-18px text
- Icon Style: Modern line icons or filled icons, consistent style throughout
- CTA Button: Different per tier (Primary for featured, secondary for others)
- Tooltip Icons: Info icons (16px) next to features, hover reveals explanation
- Border Treatment: Gradient borders for premium tiers, glow effects
- Background: Each tier gets subtle background tint or pattern
- Calculator UI: Range slider with live price update, usage visualization
- Table Layout: Sticky header on scroll, alternating row colors, expandable rows
- Hover States: Cards lift, shadows intensify, slight tilt effect (3D)
- Responsive: Stack cards vertically on mobile, maintain feature alignment
- Accessibility: High contrast prices, clear labels, keyboard-navigable toggles

**Psychological Pricing**:
- Anchor with highest price first (if beneficial)
- Show savings prominently with strikethrough
- Use $99 not $99.00 (remove unnecessary decimals)
- Annual pricing shows monthly equivalent

## 7. FAQ SECTIONS

**Component Purpose**: Address objections and questions with modern accordion that's actually enjoyable to use

**Required Variations**:
- Two-column accordion with staggered animations
- Search-powered FAQ (instant filter)
- Category tabs with nested accordions
- Video answer options for complex questions
- AI chatbot integration for dynamic answers
- Split layout (questions left, answers right with transitions)

**Design Specifications**:
- Accordion Item: 72-88px height collapsed, smooth expand animation (400ms)
- Question Typography: 20-24px, medium/semi-bold weight, left-aligned
- Answer Typography: 16-18px, line-height 1.7, max-width 700px, gray text (70% opacity)
- Icon Design: Plus/minus (28-32px) or chevron, smooth rotation animation
- Hover State: Background color change, subtle highlight, cursor pointer
- Open State: Different background color, shadow inside, expanded padding
- Spacing: 12-16px between items, 60-80px section padding
- Search Bar: Sticky at top, 56-64px height, magnifying glass icon, "Ask a question..."
- Search Results: Highlight matching text, filter accordion in real-time
- Category Tabs: Pill-style tabs, active state with background fill, smooth transition
- Video Answers: Thumbnail with play button, modal player or inline expand
- Chatbot UI: Floating chat bubble, modern chat interface, instant responses
- Split Layout: 50/50 or 40/60, click question left to show answer right
- Animation: Smooth height transitions, fade-in content, stagger items on page load
- Background: Subtle different from main page (light gray or white on colored page)
- Responsive: Single column on mobile, search remains functional
- Accessibility: Proper ARIA labels, keyboard navigation (arrow keys), focus management

**Content Strategy**:
- Front-load most important questions
- Use natural language (how users actually ask)
- Keep answers concise (under 100 words)

## UNIVERSAL MODERN LANDING PAGE PRINCIPLES

**Color Systems for Modern Landings:**
- **Tech/SaaS**: Electric blues (#0066FF), vibrant purples (#7C3AED), neon accents
- **Finance**: Deep blues (#1E40AF), greens (#059669), trust-building neutrals
- **Creative/Agency**: Bold gradients, pink (#EC4899) to orange (#F97316)
- **Wellness**: Soft pastels, sage greens (#10B981), calming blues (#3B82F6)
- **Dark Mode**: True black (#000000) or near-black (#0A0A0A) with colored accents

**Typography Scale (Modern):**
- Display/Hero: 64-96px (extra bold, dramatic)
- H1: 48-64px (bold)
- H2: 36-48px (semi-bold to bold)
- H3: 28-36px (medium to semi-bold)
- H4: 24-28px (medium)
- Body Large: 18-20px (regular)
- Body: 16-18px (regular)
- Small: 14-16px (regular)
- Tiny: 12-14px (use sparingly)

**Font Pairing Strategies:**
- Display font for headlines (geometric sans, bold serif)
- Clean sans-serif for body (Inter, SF Pro, Helvetica Now)
- Limit to 2 font families maximum
- Use font weight variations within same family

**Spacing System (8px base with golden ratio):**
